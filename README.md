> DIY aluminum profile home cage assembly guide for rodent research.

## 📖 概述

本项目提供**居家笼（Home Cage）铝型材支架**的完整安装指南，包含：

- 📦 **材料清单** — 所需全部型材、角件、螺丝规格
- 🎬 **视频教程** — 角件连接技巧 + 整体组装流程
- 🔧 **分步图文** — 4 个核心安装步骤详解
- 🧩 **旷场安装** — 旷场组件组装方法
- 💡 **关键技巧** — T 型螺母安装、预紧力控制等

## 🚀 快速开始

访问 GitHub Pages 站点：

```
https://aprsev.github.io/Home_cage_install/
```

或直接打开 `index.html` 本地浏览。

### 所需工具

| 工具 | 用途 |
|------|------|
| 内六角扳手（对应螺丝规格） | 锁紧螺丝 |
| 橡胶锤 | 微调型材位置 |
| 水平尺 | 校正框架水平 |
| 卷尺 | 测量定位 |
